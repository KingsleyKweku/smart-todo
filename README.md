# Smart To-Do 

A simple but powerful to-do list app built with HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Edit tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Due dates
- Saves data in browser storage

## Built With
- HTML
- CSS
- JavaScript
